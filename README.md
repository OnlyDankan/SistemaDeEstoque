Sistema de Controle de Estoque

Sistema de estoque desenvolvido em C# (.NET) com o objetivo de praticar Programação Orientada a Objetos, LINQ e manipulação de coleções. A aplicação permite realizar o gerenciamento de produtos por meio de uma interface em console.

Funcionalidades

- Cadastro de produtos
- Listagem de produtos
- Busca por código utilizando LINQ
- Atualização de produtos
- Remoção de produtos
- Entrada de estoque
- Saída de estoque

Tecnologias

- C#
- .NET
- Programação Orientada a Objetos (POO)
- LINQ

Estrutura


SistemaEstoque
│
├── Models
│   └── Produto.cs
│
├── Services
│   └── ProdutoService.cs
│
└── Program.cs

Objetivo

Este projeto foi desenvolvido para praticar conceitos fundamentais de C#, como POO, organização de código e manipulação de dados em memória, simulando um sistema de controle de estoque utilizado em empresas.
